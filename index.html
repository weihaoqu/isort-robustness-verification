Relational Cell Morphing
Automated verification of array-manipulating algorithms using PCSAT

How It Works

Relational Cell Morphing reduces whole-array reasoning to tracking a single symbolic "distinguished cell." Instead of modeling every element, we follow one index k through two parallel runs of the algorithm, capturing how input perturbations propagate to outputs.

1
Pick a Cell
Track one symbolic index k across two runs
2
HIT / MISS
When the algorithm accesses index j: HIT if j=k, MISS otherwise
3
Encode in PCSAT
Express invariants and transitions as constrained Horn clauses
4
Verify
UNSAT = property holds for all inputs and all array sizes
Robustness Verification
Robustness
Insertion Sort
1-robust under L-infinity: perturbing any single input element by at most ε changes any output element by at most ε. Verified via the HIT/MISS abstraction with explicit sign-bit tracking.
1-robust L∞ norm HIT / MISS
Robustness
Kruskal's Algorithm
(N−1)-robust: perturbing every edge weight by at most ε changes the MST cost by at most (N−1)·ε. Models HIT-HIT, HIT-MISS, MISS-HIT, MISS-MISS edge selection matrix.
(N−1)-robust L∞ norm HIT / MISS matrix
Robustness
Dijkstra's Algorithm
(N−1)-robust: perturbing every edge weight by at most ε changes shortest-path distances by at most (N−1)·ε. Uses hop-count abstraction with merged HIT/MISS.
(N−1)-robust L∞ norm Hop count
Sensitivity Verification
Sensitivity
Histogram
L1 sensitivity 2 under the replace metric: changing one input value shifts counts in at most two bins (one down, one up). Tracks two distinguished cells with hit1/hit2 booleans.
L1 sensitivity 2 Replace metric hit1 / hit2
Sensitivity
CDF (Cumulative Distribution)
L1 sensitivity (m−1) under the replace metric: changing one input value shifts cumulative counts for all bins above the affected range. Uses sign-bit tracking on the cumulative delta.
L1 sensitivity (m−1) Replace metric Cumulative delta
